# For Khushi 🌹

A romantic animated love card webpage with:
- Soft 3D flip animation
- Typing effect
- Floating hearts
- Elegant white rose aesthetic
- Mobile-friendly design

## Files
- index.html
- style.css
- script.js

## How to Run
Open `index.html` in your browser.

## Publish on GitHub Pages
1. Upload all files to a GitHub repository
2. Go to Settings → Pages
3. Deploy from branch → main
4. Save
